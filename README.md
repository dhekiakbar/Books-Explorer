# Books Spider Scraper

Spider to learn big data and programming lecture

# How to use
```
# Install Requirement
pip install scrapy pick

# Scrap books data
cd Books-Explorer
scrapy crawl books -o books.json

# Explorer menu
python explore-books.py

```
